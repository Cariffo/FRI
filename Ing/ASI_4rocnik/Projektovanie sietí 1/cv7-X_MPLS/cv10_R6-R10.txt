NA PE ROUTRE (1,3,4,9) - tu je ukazkova konfiguracia R9

ip vrf z1
rd 110:1
route-target 110:1

int e2/1
ip vrf forwarding z1
ip address 192.168.89.9 255.255.255.0

ping vrf z1 192.168.89.9

router bgp 110
address-family ipv4 unicast vrf z1
redistribute connected

#########################################

KONFIGURACIA INTERNEHO BGP

router bgp 110
no bgp default ipv4-unicast
neighbor 10.255.255.1 remote-as 110
neighbor 10.255.255.1 update-source lo0
address-family vpnv4
neighbor 10.255.255.1 activate

KONFIGURACIA ROUTE REFLECTOR (R1)
router bgp 110
no bgp default ipv4-unicast
neighbor 10.255.255.3 remote-as 110
neighbor 10.255.255.3 update-source lo0
neighbor 10.255.255.4 remote-as 110
neighbor 10.255.255.4 update-source lo0
neighbor 10.255.255.9 remote-as 110
neighbor 10.255.255.9 update-source lo0
address-family vpnv4
neighbor 10.255.255.3 activate
neighbor 10.255.255.4 activate
neighbor 10.255.255.9 activate
neighbor 10.255.255.3 route-reflector-client
neighbor 10.255.255.4 route-reflector-client
neighbor 10.255.255.9 route-reflector-client

#########################################

KONFIGURACIA EXTERNEHO BGP na PE (ukazka na R1)
router bgp 110
address-family ipv4 unicast vrf z1
neighbor R5 remote-as 65001
neighbor R5 activate
