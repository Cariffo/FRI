Co je GNU Emacs?

GNU Emacs je velice komplexní a sofistikovaný textový editor se zabudovaným interpretrem jazyka Emacs Lisp (jeden z dialektů jazyka LISP) pomocí něhož je snadno rozšiřitelný.
Obsahuje také vývojové prostředí mnoha programovacích jazyků, poštovní klient, webový prohlížeč, kalendář, diář, několik her, psychiatra a mnoho dalšího.
Existují verze pro mnoho operačních systémů, mimo jiné pro GNU/Linux, FreeBSD, OpenBSD, NetBSD, AIX, OpenSolaris, MacOS X, ale také pro MS Windows a DOS.
Emacs je součástí projektu GNU a je šířen pod licencí GNU GPL, což ve zkratce znamená, že je zcela zdarma včetně kompletních zdrojových kódů, je možno ho libovolně šířit a modifikovat.
Název Emacs vznikl jako zkratka pro Editor MACroS ("editorová makra"), nebot se původně jednalo pouze o sadu maker pro editor TECO.
Kromě GNU Emacsu existuje také XEmacs, verze Emacsu pro X.
